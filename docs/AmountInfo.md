# AmountInfo

## Properties

| Name         | Type       | Description                                                                 | Notes |
|--------------|------------|-----------------------------------------------------------------------------|-------|
| **currency** | **String** | Currency code for the payment. Must respect the ISO 4217 and be upper case. | -     |
| **value**    | **Number** | The total amount of the transaction in the specified currency.              | -     |

## Example

```json
{
  "amount": 500,
  "currency": "EUR"
}
```
